================================================================================
TROUBLESHOOTING
================================================================================

If login fails:
1. Verify database is imported correctly
2. Check Apache and MySQL are running
3. Clear browser cache and cookies
4. Try different browser
5. Verify credentials are typed correctly (case-sensitive)

If admin panel shows error:
1. Check database connection in config/database.php
2. Verify admin table has data
3. Check session is enabled in PHP

If user panel shows error:
1. Check database connection
2. Verify users table has data
3. Clear browser cache
